# Cryptobay Mobile App

## Run app
```
yarn ios
```

## Run Storybook
```
yarn stroybook
yarn ios:storybook
```

## Run Tests
```
yarn test
tarn test ${relativePathToComponentsTest}
```

## Run Test with Coverage
```
yarn test:coverage
```

## Rubby Version
```
3.0.0
```
