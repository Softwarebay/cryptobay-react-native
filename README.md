# Cryptobay Mobile App

Cryptobay is a mobile cryptocurrency wallet that allows you to exchange and collect currencies with your friends. 

## Run app
```
yarn ios
```

## Run Storybook
```
yarn stroybook
yarn ios:storybook
```

## Run Tests
```
yarn test
```

## Run Test with Coverage
```
yarn test:coverage
```
